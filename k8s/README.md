# Kubernetes Notu

Bu proje Docker Compose ile hizli demo icin hazirlandi. Cloud-native mimariyi gostermek icin ayni bilesenler Kubernetes ortaminda su sekilde kurgulanabilir:

- `metric-app`: Deployment + Service
- `prometheus`: ConfigMap + Deployment + Service
- `grafana`: ConfigMap provisioning + Deployment + Service
- `alertmanager`: ConfigMap + Deployment + Service

Sunumda "Docker Compose yerel demo, Kubernetes ise production benzeri dagitim modeli" seklinde anlatilabilir.
